BAF (Bazaar Auction Flipper) is a headless automation client for Hypixel Skyblock with configurable workflows, cross-platform builds, and operator-focused diagnostics. This page highlights macro capabilities and managed hosting options.

Open source on GitHub

Use at your own risk: according to the project documentation, this can violate Hypixel Terms of Service and may lead to bans.

Macro features

  • Automatic buy/sell flipping flow with account authentication and island routing.
  • Skip-confirmation controls based on profit, finder source, skins, and price thresholds.
  • Automatic account switching with timed rotation across multiple usernames.
  • SOCKS5 proxy support with optional username/password authentication.

Operations & observability

  • Webhook notifications for init, purchasing, selling, and relisting events.
  • Bazaar flip debugging logs for message parsing, GUI state, and action traces.
  • Config-driven startup via config.toml with runtime persistence.
  • Release pipeline provides pre-built executables for Windows, Linux, and macOS.
