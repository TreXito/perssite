Frikadellen BAF

BAF (Bazaar Auction Flipper) is a headless macro client for Hypixel Skyblock.

GitHub

Use at your own risk: the macro violates Hypixels TOS and can get you banned

Macro features

  • Authentication flow, then automated buy/sell flipping after joining Hypixel and teleporting to island.
  • Skip-confirmation controls in config.toml using profit, finder source, skins, and price thresholds.
  • Automatic account switching with timed rotation across configured usernames.
  • SOCKS5 proxy support including optional username/password authentication.

Operations & observability

  • Webhook notifications for init, purchasing, selling, and relisting events.
  • Bazaar flip debugging logs for message parsing, GUI state, and action traces.
  • Config-driven startup via config.toml (username, skip config, proxy config, account switching, webhook URL).
  • Pre-built executables for Windows, Linux, and macOS in GitHub Releases.
